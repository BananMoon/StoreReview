# StoreReview
리뷰 서비스를 제공하는 웹 사이트 개발 프로젝트입니다. 😊
## 팀원 🧑🏻‍🤝‍🧑🏻
- Front-End : 이수경
- Back-End : 문윤지, 조준희

## 주요 기능 💃
- 이미지 업로드, 별점, 댓글, 로그인, 회원가입, 검색, 가게 정보 조회

## 기술 스택🔧

### Front-End
- **Library**
    - react-router-dom
    - react
    - react-redux
- HTML5
- CSS
- Javascript

### Back-End
- **Server & Library**
    - Java 8
    - Spring Boot 2.5.x
    - Spring Data JPA + querydsl
- **Build Tool**
    - Gradle
- **DataBase**
    - MySQL
- **Infra**
    - AWS EC2
    - AWS RDS
    - AWS S3
